== Description ==

When adding media on the post screen, automatically add loading = "lazy" to the img tag.

You don't have to write loading = "lazy" manually.

== Installation ==

1. From the WP admin panel, click “Plugins” -> “Add new”.
2. In the browser input box, type “loading-lazy”.
3. Select the “My Custom Style Css Manager” plugin and click “Install”.
4. Activate the plugin.
